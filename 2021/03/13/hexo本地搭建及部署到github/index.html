hexo本地搭建及部署到github

三月 13, 2021

新建github存储库

打开github,打开新建存储库，存储库名称输入(你喜欢的英文).github.io/com
点击创建

环境

首先我们需要下载环境，需要拥有两个环境：

nodejs

nodejs的网址：nodejs.org

安装完nodejs后，win+R然后打开cmd，输入node -v回车

git for windows

gitforwindows:gitforwindows.org

在这里安装向导省略
然后win+R打开cmd,输入git回车

githubSHH的配置

在git的安装目录下打开git bash
(如果找不到，搜索ssh)
输入ssh-keygen -t rsa -c "你的邮箱"回车再回车

网页打开github,点击头像/settings/ssh ang GPGkeys
在title栏随便写，打开用户/.ssh/id_rsa.pub,将内容复制到Key。
点击Add SSH key。
输入ssh -T git@giyhub.com可验证是否成功

hexo的本地使用

在硬盘中新建一个文件夹，然后win+R打开cmd，cd到新建文件夹下
输入npm install hexo-cli -g回车，等待安装完成。
完成后继续在cmd上输入hexo init 你的博客名回车。
完成后继续输入hexo init回车

安装完成后输入hexo s -p 5555回车
在浏览器输入localhost:5555
然后我们就可以看到我们的本地hexo啦。

sublime安装

为了便于hexo的编写，我们需要下载sublime

将本地hexo部署到远端github

在目录文件夹上打开git bash,hexo s开启本地服务
复制粘贴打开窗口

回到git bash，ctrl+c然后输入git config --global user.name '你的用户名'
再输入你的邮箱git config --global user.email '你的邮箱'
打开根目录，找到最下方的
type: git 完成

回到github仓库，复制创建仓库的ssh到repo：
repo:以上

再加上
branch: master
ctrl+s保存

最终部署

我们回到git bash 输入hexo g生成文件
再输入hexo d
这里部署出错的话输入npm install hexo-deployer-git --save安装插件即可。
